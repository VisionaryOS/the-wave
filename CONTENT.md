## Technical Implementation Notes

### HTML5 Semantic Elements Used
- `<header>` - Site header and navigation
- `<nav>` - Primary navigation
- `<main>` - Main content area
- `<article>` - Individual news articles
- `<section>` - Logical content sections
- `<figure>` / `<figcaption>` - Images with captions
- `<footer>` - Site footer
- `<time>` - Article dates with datetime attribute

### CSS Features Used
- CSS Custom Properties (variables)
- CSS Grid and Flexbox layouts
- CSS-only wave transition (border-radius, pseudo-elements)
- Glassmorphism effect (backdrop-filter)
- CSS hover animations
- Responsive media queries
- Print stylesheet
- Focus-visible for accessibility

### Accessibility Features
- Skip link for keyboard navigation
- Semantic HTML structure
- ARIA labels where needed
- Focus-visible styles
- Color contrast compliance
- Descriptive alt text (via aria-label on placeholders)
