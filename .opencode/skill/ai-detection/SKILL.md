# AI Detection Skill

Patterns that reveal code was generated by AI rather than written by a human.

## Scoring System

| Category | Weight | Threshold |
|----------|--------|----------|
| Conversational residue | 3x | Instant flag |
| Placeholder content | 3x | Instant flag |
| Perfect formatting | 2x | Suspicious |
| Generic naming | 1.5x | Note |
| Explicit defaults | 1.5x | Note |
| Over-documentation | 1x | Note |

**Score > 5 = Likely AI-generated**

## High Confidence Indicators

### Conversational Residue (Smoking Gun)
```html
<!-- INSTANT FLAG -->
<!-- As you can see, this section contains... -->
<!-- Let me explain how this works -->
<!-- Here is the updated code: -->
<!-- I've added a responsive layout -->

<!-- Human equivalent -->
<!-- hero section -->
<!-- responsive grid -->
<!-- (or no comment at all) -->
```

### Placeholder Content
```html
<!-- INSTANT FLAG -->
<img src="https://via.placeholder.com/300">
<a href="#">Link</a>
<p>Lorem ipsum dolor sit amet...</p>
<input value="your_email@example.com">

<!-- Human equivalent -->
<img src="/assets/hero.jpg" alt="Product screenshot">
<a href="/pricing">View pricing</a>
<p>Ship faster with AI agents.</p>
```

### Perfect Formatting
```css
/* SUSPICIOUS: Zero drift across entire file */
.card {
    display: flex;
    flex-direction: column;
    padding: 1rem;
}

.button {
    display: flex;
    align-items: center;
    padding: 0.5rem 1rem;
}

/* Human: Shows evolution/drift */
.card {
    display: flex;
    flex-direction: column;
    padding: 1rem;
}

.button {
  display: flex;  /* different indent style */
  align-items: center;
  padding: .5rem 1rem;  /* .5 not 0.5 */
}
```

## Medium Confidence Indicators

### Textbook Comments
```css
/* AI: Explains WHAT (redundant) */
/* This CSS rule sets the display property to flex */
.container { display: flex; }

/* Human: Explains WHY */
/* flexbox for IE11 compat, would use grid otherwise */
.container { display: flex; }
```

### Generic Naming
```html
<!-- AI: Overly literal/generic -->
<div class="main-content-wrapper-container">
  <div class="user-profile-information-section">
    <div class="profile-image-container">

<!-- Human: Contextual/abbreviated -->
<div class="content">
  <section class="profile">
    <div class="avatar">
```

### Explicit Browser Defaults
```css
/* AI: Declares what browser already does */
div {
  display: block;
  position: static;
  float: none;
  box-sizing: content-box;
}

/* Human: Only what's needed */
div {
  /* empty - it's a div */
}
```

## Low Confidence Indicators

### Structural Boilerplate
```html
<!-- AI: Over-structured -->
<div class="page-wrapper">
  <div class="content-container">
    <div class="main-content-area">
      <div class="section-wrapper">
        <div class="inner-content">
          <p>Hello</p>

<!-- Human: Direct -->
<main>
  <p>Hello</p>
```

### Missing Human Artifacts
- No `TODO` or `FIXME` comments
- No commented-out code
- No browser-specific hacks
- No evidence of refactoring
- No "weird" edge case handling
- No personal abbreviations

### Perfect BEM Everywhere
```css
/* AI: Textbook perfect */
.navigation__list-item--active { }
.user-profile__avatar-image--large { }

/* Human: Shortcuts */
.nav-item.active { }
.avatar.lg { }
```

## HTML-Specific Tells

### AI Pattern
```html
<!-- Rigid structure even for simple content -->
<header class="header-container">
  <nav class="navigation-wrapper">
    <ul class="navigation-list">
      <li class="navigation-list-item">
        <a class="navigation-link" href="#">Home</a>
```

### Human Pattern
```html
<header>
  <nav>
    <a href="/">Home</a>
    <a href="/about">About</a>
```

## CSS-Specific Tells

### AI Pattern
```css
/* Over-specified, safe choices */
.button {
  display: inline-flex;
  align-items: center;
  justify-content: center;
  padding: 0.75rem 1.5rem;
  font-size: 1rem;
  font-weight: 500;
  line-height: 1.5;
  color: #ffffff;
  background-color: #007bff;
  border: 1px solid transparent;
  border-radius: 0.375rem;
  cursor: pointer;
  text-decoration: none;
  transition: all 0.15s ease-in-out;
}
```

### Human Pattern
```css
/* Minimal, relies on context */
.btn {
  padding: var(--space-2) var(--space-4);
  background: var(--color-primary);
  border-radius: var(--radius-md);
}
```

## Detection Checklist

### Instant Flags
- [ ] "As you can see" / "Let me explain"
- [ ] `placeholder.com` or `example.com`
- [ ] `href="#"` on multiple links
- [ ] Lorem ipsum text
- [ ] Perfect consistent formatting

### Suspicious
- [ ] Comments explaining syntax
- [ ] `.container`, `.wrapper` class names
- [ ] `z-index: 9999`
- [ ] Explicit browser defaults
- [ ] Full BEM on everything

### Investigate
- [ ] No TODOs or FIXMEs
- [ ] No commented-out code
- [ ] No abbreviations
- [ ] Over-structured HTML
- [ ] Every element has a class
